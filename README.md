# EigenPuzzle

## 运行环境

* c++17, gcc>=8.0 或者 clang


## 简要说明

`Puzzle.hpp` 里定义里问题，`Solver.hpp` 里为每一个算法的实现，`main.cpp`里
提供两个测试函数，分别进行固定的几个简单测试和200大小的随机序列测试

在运行时，注意去掉/加上注释
